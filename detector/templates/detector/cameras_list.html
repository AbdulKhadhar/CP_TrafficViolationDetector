How to Use Live Monitoring
  1. Add Camera: Go to Admin Panel â†’ Cameras and add a new camera with RTSP URL or leave blank for USB webcam
  2. Start Detection: Click "Start Detection" button on the camera card
  3. View Live Feed: Click "View Live Feed" to see real-time monitoring
  4. Review Violations: Detected violations appear automatically in the dashboard
  5. Stop Detection: Click "Stop Detection" when done
Camera Setup Examples:
  • USB Webcam: Leave RTSP URL blank, system will use cv2.VideoCapture(0)
  • IP Camera (Hikvision): rtsp://username:password@192.168.1.100:554/Streaming/Channels/101
  • IP Camera (Dahua): rtsp://username:password@192.168.1.100:554/cam/realmonitor?channel=1
  • Generic RTSP: rtsp://camera-ip:554/stream
